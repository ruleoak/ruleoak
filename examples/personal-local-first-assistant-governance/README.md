# Personal Local-First Assistant Governance

Run:

```bash
npm run personal:local-assistant-governance
```

This reference vertical shows how RuleOak can govern a privacy-first personal assistant. Local note/calendar reads and drafting are allowed, external sends require approval, and private-file upload is blocked by default.

Outputs are written to `out/`: governance records, evidence bundle, audit log, approval request, JSON report, and HTML report.
