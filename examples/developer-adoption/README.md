# Developer Adoption Examples

These examples show the shortest integration paths for application developers.

They are separate from the richer reference verticals. The goal is clarity:

- wrap an existing tool without redesigning the app;
- use the same governance pattern with LangGraph-style tools;
- use the same governance pattern with CrewAI-style tools;
- see how an MCP-local-proxy boundary fits into the same decision model.

Run:

```bash
npm run adoption:real-frameworks
```
