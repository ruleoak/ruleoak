Safe single-file HTML timeline

ruleoak replay --html .ruleoak/report.html creates a local evidence timeline from hash-chained Evidence JSONL.

The generated report is self-contained: no Tailwind CDN, no remote scripts, no external fonts, and no network resources.

Every evidence-derived field is HTML-escaped before interpolation, including action names, matched rules, decision reasons, hashes, metadata, and tool arguments. This matters because poisoned tool arguments may contain strings such as <script>alert(1)</script>. The report must show that as text, not execute it.
