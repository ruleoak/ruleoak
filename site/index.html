Agent Firewall + Flight Recorder for AI Agents.

Your AI agent can delete files, run shell commands, and call tools now. RuleOak blocks dangerous JSON-RPC tool calls on the inbound stream before the tool server sees them, then gives you a black-box recording of every action.

What changed in the security path

Block before forward

RuleOak now evaluates JSON-RPC tools/call requests on the client → server stdin path before forwarding them to the child process.

Safe to open offline

HTML reports use inline CSS only and escape evidence-derived fields, including adversarial tool arguments.

Honest boundary

RuleOak is a stdio/JSON-RPC stream gate. It does not claim kernel-level syscall interception or full MCP gateway support yet.
