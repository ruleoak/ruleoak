# @ruleoak/protocol

MIT-licensed standalone protocol package for RuleOak-compatible action envelopes, policy files, and evidence JSONL.

Use this package when you want to emit or validate RuleOak-compatible evidence without depending on RuleOak Core.

```js
import { validateActionEnvelope, validatePolicy } from '@ruleoak/protocol';
```
