# GitHub Repository Settings

Recommended public GitHub settings for RuleOak Core.

## Enable

- Issues
- Discussions, if you can monitor them
- Security policy
- Dependabot alerts
- Branch protection for `main`
- Require CI before merging
- Tags and GitHub releases

## Keep optional for now

- Sponsorship
- Wiki
- Projects board

## Do not enable until ready

- Broad roadmap promises that imply paid support
- Public package publishing automation without manual review
- Auto-release from every tag
