# Compatibility Badges

RuleOak includes badge assets for implementations that pass the relevant conformance checks.

```text
protocol-conformance-kit/badges/governance-v1-compatible.svg
protocol-conformance-kit/badges/evidence-bundle-v1-compatible.svg
protocol-conformance-kit/badges/policy-pack-v1-compatible.svg
```

Use these carefully:

| Badge | Minimum expectation |
|---|---|
| Governance v1 compatible | Golden records, strict validation, canonical JSON, and hash tests pass. |
| Evidence Bundle v1 compatible | Evidence bundle replay and tamper rejection pass. |
| Policy Pack v1 compatible | Policy-pack manifest validation, scenario tests, and compatibility checks pass. |

Recommended wording:

> Compatible with RuleOak Governance Protocol v1.

Avoid wording such as certified, regulator-approved, or independently audited unless you have separate evidence.
