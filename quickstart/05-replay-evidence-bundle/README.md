# Quickstart 05 — Replay an evidence bundle

```bash
npm run quickstart:05
npm run protocol:replay quickstart/out/05-replay-evidence-bundle/evidence-bundle.json
npm run protocol:replay quickstart/out/05-replay-evidence-bundle/audit-chain.json
```

This example writes a protocol-compatible evidence bundle and an append-only audit chain, then verifies both. It is the shortest path to understanding RuleOak Governance Protocol v1 replay.
